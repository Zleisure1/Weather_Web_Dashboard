Summary: Latitude vs. Weather Parameters

The purpose of this project was to analyze various weather parameters with changes in latitude. To accomplish this analysis, we used OpenWeatherMap API to query data from approximately 500 cities.


After collecting the data, we used MatplotLib to plot various apsects of weather vs. latitude including temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of trends and correlations.
